Introduction

Welcome to the Web Basics documentation.

This guide covers the fundamental technologies needed to build a website.

  • HTML for structure.
  • CSS for presentation.
  • JavaScript for interactivity.
console.log('Welcome to Web Development!');
HTML Fundamentals

HTML is the standard markup language for documents designed to be displayed in a web browser.

Always use semantic elements to structure your content properly.

  • Use <header> for the top section.
  • Use <main> for the primary content.
<div class="container">Hello World</div>
CSS Styling

Cascading Style Sheets (CSS) is used to style and layout web pages.

You can use CSS to alter the font, color, size, and spacing of your content.

body { background-color: #f4f4f4; color: #333; }
JavaScript Basics

JavaScript is a programming language that allows you to implement complex features on web pages.

It enables interactive web pages and is an essential part of web applications.

function greet(name) { return "Hello, " + name; }
Accessibility

Web accessibility means that websites, tools, and technologies are designed and developed so that people with disabilities can use them.

Proper contrast, semantic tags, and ARIA roles are vital components.

<button aria-label="Close menu">X</button>
